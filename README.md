# Welcome to UN

An interactive laboratory website for exploring chemistry and reactions.

## Local Development

To run the AI chatbot locally:

1. Install dependencies: `npm install` or `bun install`
2. Run both servers together: `npm run dev`
3. Open http://localhost:8082

If you need to run the API server separately, use:
- `npm run server`
- then in another terminal, `npm run dev`

## Deployment to Vercel

The AI chatbot uses a custom API endpoint and should work out of the box on Vercel.

## Features

- Interactive Chemistry Lab
- Physics Simulations
- AI-powered Science Tutor Chatbot
- Responsive design for all devices
